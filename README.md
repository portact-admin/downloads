# DSQUARE APPS — Downloads

Public release hosting for the [DSQUARE APPS](https://www.dsquareapps.com) installers.
The download buttons on the website link to the assets attached to the **latest release** here.

| App | File |
|-----|------|
| PAM (Android) | `PAM-v1.0.apk` |
| PortAct desktop (macOS) | `PortAct.pkg` |
| PortAct desktop (Windows) | `PortAct-Setup.exe` |
| PortAct Mobile (Android) | `portact-mobile-1.7.apk` |

App source lives in private repos; this repo only distributes the built installers.
